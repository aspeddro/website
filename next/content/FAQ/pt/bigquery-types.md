question: O que são os tipos no BigQuery?

São os tipos de dados do Google Standard SQL. Suas categorias incluem: INTEGER (Inteiro), STRING (Texto), DATE (Data), FLOAT64 (Decimal), GEOGRAPHY (Geográfico), entre outras.
